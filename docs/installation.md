# Installation

## Stable release

To install Personal AI System, run this command in your terminal:

```sh
uv add personal_ai_system
```

Or if you prefer to use `pip`:

```sh
pip install personal_ai_system
```

## From source

The source files for Personal AI System can be downloaded from the [Github repo](https://github.com/Natasha3101/personal_ai_system).

You can either clone the public repository:

```sh
git clone https://github.com/Natasha3101/personal_ai_system
```

Or download the [tarball](https://github.com/Natasha3101/personal_ai_system/tarball/main):

```sh
curl -OJL https://github.com/Natasha3101/personal_ai_system/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd personal_ai_system
uv sync
```
